# React E-commerce Frontend

A responsive e-commerce frontend application built using **React** and **Tailwind CSS**.  
The project focuses on frontend concepts such as component-based architecture, state management, API integration, and clean UI design.

---

##  Features

 Product listing using external JSON API  
 Add to Cart functionality  
 Global cart state management using React Context API  
 Responsive UI built with Tailwind CSS  
 Reusable components and clean folder structure  

---

## Tech Stack

 React
 Tailwind CSS
 Context API
 Vite
 JavaScript (ES6+)
